# CLAUDE.md

Project guidance for AI agents working in OpenKlip.

## Design System

Always read `DESIGN.md` before making any visual or UI decisions.
All font choices, colors, spacing, icons, and aesthetic direction are defined there.
Icon imports go through `@/lib/icon` (Phosphor fill); see the Icons section in `DESIGN.md`.
Do not deviate without explicit user approval.
In QA mode, flag any code that doesn't match `DESIGN.md`.

## Agent workflow

See `AGENTS.md` for the full OpenKlip edit loop, CLI commands, MCP tools, and guardrails.
